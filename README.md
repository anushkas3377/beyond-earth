# 🚀 Beyond Earth — Space Exploration Platform

> Explore planets, travel through time, and experience the universe like never before.

## 🌍 Live Demo
[beyond-earth-frontend.vercel.app](https://beyond-earth-frontend.vercel.app)

## ✨ Features
- 🪐 **Explore Planets** — Detailed info on all solar system planets
- 👨‍🚀 **Astronauts** — Profiles of legendary space explorers
- ⏱️ **Timeline** — Greatest milestones in space history
- 🚀 **Mission Builder** — Build and launch your own space mission
- 🔬 **Interactive Lab** — Real physics formulas and calculations
- 🏢 **Space Agencies** — NASA, ISRO and more
- 📱 **Fully Responsive** — Works on mobile and desktop

## 🛠️ Tech Stack
- **Frontend** — React 19 + TypeScript
- **Styling** — Tailwind CSS + Framer Motion
- **Routing** — TanStack Router
- **Build Tool** — Vite
- **Deployment** — Vercel

## 🚀 Getting Started
```bash
cd src/frontend
pnpm install
pnpm dev
```
